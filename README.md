# WeatherBuddy
## Purpose
This is an app designed to return the current weather and a 5 day forcast based upon a city input!

## Built With
* HTML
* CSS
* JavaScript

## Additional Tools
* jQuery
* Bootstrap
* moment.js
* OpenWeatherMap API
* MapBox Geocoding API
